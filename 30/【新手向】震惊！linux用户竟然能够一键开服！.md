这是一个较为新手向的开服教程贴
废话不多说
登录你的root账户，来到/root目录
```
cd /root
```
运行
```
apt-get install wget&&wget -P /root https://raw.githubusercontent.com/DazeCake/onebds/master/onebds.sh&&chmod +rx onebds.sh&&./onebds.sh
```
之后跟着操作就可以快速开服和部署bdl for linux
基本上3分钟就可以部署完毕
